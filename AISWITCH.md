# 🚀 AI Switch - ใช้งานง่ายที่สุด!

## 🎯 แนวคิด

**เลือก AI ด้วยคำสั่งง่ายๆ** → **พิมพ์ข้อความ** → **ได้ผลลัพธ์**

ไม่ต้องรู้เรื่องเทคนิค ไม่ต้อง config อะไรเยอะ!

---

## 📦 ติดตั้ง

```cmd
ais
```

หรือ

```cmd
aiswitch
```

---

## 💡 วิธีใช้

### 1. เริ่มโปรแกรม
```cmd
C:\> ais

╔════════════════════════════════════════════════════════╗
║               🤖 AI Switch - เลือก AI ง่ายๆ           ║
╠════════════════════════════════════════════════════════╣
║  ✅ /qwen     - เร็วสุด  (10-40s)  ◄ YOU ARE HERE ║
║     /kimi     - ภาษาไทย  (45-80s)                 ║
║     /codex    - เก่งโค้ด (15-25s)                 ║
║     /gemini   - ฉลาด     (120s+)                 ║
╠════════════════════════════════════════════════════════╣
║  /qwen /kimi /codex /gemini /all - เลือก AI     ║
║  /all - ใช้ทุกตัว | /exit - ออก                   ║
╚════════════════════════════════════════════════════════╝

📌 Using: QWEN
```

### 2. เลือก AI
```
[qwen] ▶ /kimi
```

### 3. พิมพ์ข้อความ
```

✓ KIMI (45.2s)
────────────────────────────────────────────
สวัสดีครับ! มีอะไรให้ช่วยวันนี้?
```

### 4. เปลี่ยน AI
```
[kimi] ▶ /qwen
```

### 5. ใช้ทุกตัวพร้อมกัน

## 📋 คำสั่งทั้งหมด

### เลือก AI
| คำสั่ง | AI | ความเร็ว | เหมาะสำหรับ |
|--------|-----|----------|------------|
| `/qwen` | Qwen | 10-40s ⭐⭐⭐⭐⭐ | ใช้งานทั่วไป |
| `/kimi` | Kimi | 45-80s ⭐⭐⭐ | ภาษาไทย |
| `/codex` | Codex | 15-25s ⭐⭐⭐⭐ | เขียนโค้ด |
| `/gemini` | Gemini | 120s+ ⭐ | งานซับซ้อน |
| `/all` | ทุกตัว | - | เปรียบเทียบ |

### ระบบ
| คำสั่ง | คำอธิบาย |
|--------|----------|
| `/exit`, `/quit`, `q` | ออก |
| `/all` (พร้อมข้อความ) | ใช้ทุกตัวพร้อมกัน |

## 💡 ตัวอย่างการใช้งาน

### ใช้งานทั่วไป
```cmd
ais
/qwen
ช่วยเขียน Python function
```

### เปรียบเทียบผลลัพธ์
```cmd
ais
/all
สวัสดี

✓ QWEN (10.5s) - สวัสดีครับ!
✓ KIMI (45.3s) - สวัสดีครับ มีอะไรให้ช่วย?
✓ CODEX (25.2s) - Hello! How can I help?
✓ GEMINI (120.5s) - Hi there! What can I do for you?
```

### เขียนโค้ด
```cmd
ais
/codex
เขียน function factorial

✓ CODEX (25.3s)
────────────────────────────────────────────
def factorial(n):
    if n <= 1:
        return 1
    return n * factorial(n - 1)
```

### ถามภาษาไทย
```cmd
ais
/kimi
อธิบาย list comprehension

✓ KIMI (45.2s)
────────────────────────────────────────────
List comprehension คือวิธีสร้าง list แบบสั้นใน Python...
```

## ⌨️ Workflow แนะนำ

### แบบเร็ว (แนะนำ)
```
ais → /qwen → <prompt> → ได้ผลลัพธ์
```

### แบบเปรียบเทียบ
```
ais → /all → <prompt> → เปรียบเทียบ 4 คำตอบ
```

### แบบสลับ AI
```
ais
/qwen → <prompt 1>
/kimi → <prompt 2>
/codex → <prompt 3>
/exit
```

## 🐛 Troubleshooting

### "Command not found"
```cmd
python C:\Users\user\multi-ai-agent\src\aiswitch.py
```

### "Agent ไม่พร้อมใช้"
บาง agent อาจปิดอยู่ แก้ใน `src/config.json`:
```json
"qwen": {"enabled": true}
```

### ช้าเกินไป
```cmd
ais
/qwen      # ใช้ตัวเร็วสุด
<prompt>
```
